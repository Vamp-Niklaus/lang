Forward List (std::forward_list)

Singly-linked list. Memory efficient but can only be iterated forward.

Category: Sequence Container

Overview

#include <forward_list>

std::forward_list<int> fl;

fl.push_front(10);
fl.pop_front();
fl.insert_after(fl.begin(), 20);
fl.erase_after(fl.begin());

// Note: No push_back or pop_back! No size()!

Java does not provide a singly-linked list in the standard library. You can use LinkedList (which is doubly-linked) or implement your own.

Python does not provide a singly-linked list. For competitive programming or specific uses, you must define a custom class.

class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

JavaScript does not provide a singly-linked list. You must define a custom class.

class ListNode {
    constructor(val, next = null) {
        this.val = val;
        this.next = next;
    }
}
