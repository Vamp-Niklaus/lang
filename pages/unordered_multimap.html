Unordered Multimap

Hash table map that allows duplicate keys.

Category: Unordered Container

Overview

#include <unordered_map>
std::unordered_multimap<int, std::string> umm;

umm.insert({1, "One"});
umm.insert({1, "Uno"});

// operator[] is not available
auto range = umm.equal_range(1);

🌎 Polyglot Equivalents

  • ☕
    Java: java.util.HashMap<K, List<V>>
  • 🐍
    Python: collections.defaultdict(list)
  • 🟨
    JavaScript: Map<K, Array<V>>
