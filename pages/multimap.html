Multimap (std::multimap)

Map that allows duplicate keys.

Category: Associative Container

Key Differences from Map

Because keys can be duplicated, operator[] is not available.

#include <map>
std::multimap<int, std::string> mm;

// Must use insert with a pair
mm.insert({1, "One"});
mm.insert({1, "Uno"}); // Duplicate key allowed!

// mm[1] = "Eins"; // ERROR!

// Find elements
auto range = mm.equal_range(1);
for (auto it = range.first; it != range.second; ++it) {
    // "One", "Uno"
}
