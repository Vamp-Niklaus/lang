Multimap (std::multimap)

Map that allows duplicate keys.

Category: Associative Container

Key Differences from Map

Because keys can be duplicated, operator[] is not available.

#include <map>
std::multimap<int, std::string> mm;

// Must use insert with a pair
mm.insert({1, "One"});
mm.insert({1, "Uno"}); // Duplicate key allowed!

// mm[1] = "Eins"; // ERROR!

// Find elements
auto range = mm.equal_range(1);
for (auto it = range.first; it != range.second; ++it) {
    // "One", "Uno"
}

TreeMap<K, List<V>>

Map a single key to a List of values.

collections.defaultdict(list)

The exact equivalent of a multimap in Python is mapping a Key to a List of Values using a defaultdict.

from collections import defaultdict

multimap = defaultdict(list)

# Insert multiple values for same key
multimap["A"].append(1)
multimap["A"].append(2)
multimap["B"].append(3)

# Iterate over all values for key "A"
for val in multimap["A"]:
    print(val) # Prints 1, then 2

Map of Arrays

In JavaScript, a multimap is easily constructed by mapping a Key to an Array of Values.

const multimap = new Map();

function insert(key, val) {
    if (!multimap.has(key)) {
        multimap.set(key, []);
    }
    multimap.get(key).push(val);
}

insert("A", 1);
insert("A", 2);

// Retrieve
let values = multimap.get("A"); // [1, 2]
