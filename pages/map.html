Map (std::map)

Implemented as a balanced binary search tree (Red-Black Tree).

  • Insert/Delete/Search by key: O(log n)
#include <map>

std::map<int, int> m;
m[1] = 2;          // Insert or update a key-value pair - O(log n)
m.emplace(1, 2);   // Insert a key-value pair - O(log n)
m.find(1);         // Find an element by key - O(log n)
m.erase(1);        // Remove an element by key - O(log n)
m.clear();         // Remove all elements - O(n)
m.size();          // Get the number of elements - O(1)

TreeMap<K, V>

java.util.TreeMap is the exact equivalent of std::map. It is a Red-Black tree based NavigableMap implementation. The map is sorted according to the natural ordering of its keys, or by a Comparator provided at map creation time.

Method Mapping

  • operator[k] = v → put(k, v)
  • erase(k) → remove(k)
  • find(k) != end() → containsKey(k)
  • lower_bound(k) → ceilingKey(k)

Time Complexities

  • Search (get)O(log N)
  • Insert (put)O(log N)
  • Delete (remove)O(log N)
import java.util.TreeMap;
import java.util.Map;

TreeMap<Integer, String> map = new TreeMap<>();

// Insert O(log n)
map.put(3, "C");
map.put(1, "A");
map.put(2, "B");

// Iterating perfectly sorted by Key
for (Map.Entry<Integer, String> entry : map.entrySet()) {
    System.out.println(entry.getKey() + ": " + entry.getValue());
}

// Access O(log n)
String val = map.get(2); // Returns "B"

// Bounds
Integer lb = map.ceilingKey(2); // Returns 2

// Remove O(log n)
map.remove(1);

dict (Insertion Ordered)

Python's dict is a hash map, but since Python 3.7 it perfectly preserves insertion order. If you strictly need keys to be sorted like std::map, you must sort the keys on demand or use a sorted list of tuples.

# 1. Standard dict (O(1) insert/lookup)
d = {}
d[3] = "C"
d[1] = "A"
d[2] = "B"

# 2. Emulating std::map sorted iteration
for k in sorted(d.keys()):
    print(f"{k}: {d[k]}")
# Outputs: 1: A, 2: B, 3: C

Map (Insertion Ordered)

JavaScript's Map is a hash map that preserves insertion order. Like Python, if you need sorted iteration like C++'s Red-Black tree map, you must sort the keys array manually.

const map = new Map();

// Insert O(1)
map.set(3, "C");
map.set(1, "A");
map.set(2, "B");

// Emulating std::map sorted iteration
const sortedKeys = Array.from(map.keys()).sort((a, b) => a - b);
for (let k of sortedKeys) {
    console.log(k + ": " + map.get(k));
}
