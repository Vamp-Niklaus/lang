Map (std::map)

Implemented as a balanced binary search tree (Red-Black Tree).

  • Insert/Delete/Search by key: O(log n)
#include <map>

std::map<int, int> m;
m[1] = 2;          // Insert or update a key-value pair - O(log n)
m.emplace(1, 2);   // Insert a key-value pair - O(log n)
m.find(1);         // Find an element by key - O(log n)
m.erase(1);        // Remove an element by key - O(log n)
m.clear();         // Remove all elements - O(n)
m.size();          // Get the number of elements - O(1)

🌎 Polyglot Equivalents

  • ☕
    Java: java.util.TreeMap<K, V> (Sorted keys)
  • 🐍
    Python: No built-in sorted map. Standard dict preserves insertion order.
  • 🟨
    JavaScript: No built-in sorted map. Standard Map preserves insertion order.
