Set (std::set)

Implemented as a balanced binary search tree (Red-Black Tree).

  • Insert/Delete/Search: O(log n)
#include <set>

std::set<int> s;
s.insert(1); // Insert an element - O(log n)
s.find(1);   // Find an element - O(log n)
s.erase(1);  // Remove an element - O(log n)
s.clear();   // Remove all elements - O(n)
s.size();    // Get the number of elements - O(1)

🌎 Polyglot Equivalents

  • ☕
    Java: java.util.TreeSet<T> (Red-Black tree, sorted)
  • 🐍
    Python: No built-in sorted set. (Standard set is a hash table like unordered_set)
  • 🟨
    JavaScript: No built-in sorted set. (Standard Set preserves insertion order, not sorted).
