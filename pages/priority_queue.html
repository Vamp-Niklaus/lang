Priority Queue (std::priority_queue)

Implemented as a heap.

  • Insert/Delete Max: O(log n)
  • Get Max: O(1)
#include <queue>

std::priority_queue<int> pq;
pq.push(10); // Insert an element - O(log n)
pq.pop();    // Remove the top element - O(log n)
pq.top();    // Get the top element - O(1)
pq.size();   // Get the number of elements - O(1)
pq.empty();  // Check if the priority queue is empty - O(1)

PriorityQueue<T>

java.util.PriorityQueue is implemented as a Priority Heap. Note: Unlike C++ which defaults to a Max-Heap, Java defaults to a Min-Heap. To get a Max-Heap, you must pass Collections.reverseOrder().

import java.util.PriorityQueue;
import java.util.Collections;

// MIN-HEAP (Default)
PriorityQueue<Integer> minHeap = new PriorityQueue<>();

// MAX-HEAP (Like C++ default)
PriorityQueue<Integer> maxHeap = new PriorityQueue<>(Collections.reverseOrder());

// O(log N) Push
maxHeap.add(10);
maxHeap.add(20);

// O(1) Top
int top = maxHeap.peek(); // 20

// O(log N) Pop
maxHeap.poll(); // removes 20

heapq

Python provides the heapq module which operates on a standard list. Like Java, it implements a Min-Heap by default. To simulate a Max-Heap, multiply numeric values by -1 before pushing.

import heapq

# MIN-HEAP (Default)
min_heap = []
heapq.heappush(min_heap, 10)
heapq.heappush(min_heap, 20)
top = min_heap[0] # 10
heapq.heappop(min_heap)

# MAX-HEAP (Simulated with negatives)
max_heap = []
heapq.heappush(max_heap, -10)
heapq.heappush(max_heap, -20)

top = -max_heap[0] # 20

Custom Binary Heap / Sort

JavaScript lacks a built-in Heap. For smaller datasets, pushing to an array and re-sorting is easy but O(N log N). For performance, you must write a class.

// Easy but slow O(N log N) approach:
let pq = [];
pq.push(10);
pq.push(5);
pq.sort((a, b) => a - b); // Min-Heap simulation
let top = pq.shift(); // 5

// Fast O(log N) approach requires a full class:
class MinHeap {
    constructor() { this.heap = []; }
    push(val) {
        this.heap.push(val);
        this._bubbleUp(this.heap.length - 1);
    }
    pop() {
        // swap root with last, pop last, bubble down root
    }
    // ... custom bubble up / down logic
}
