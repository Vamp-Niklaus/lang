Binary Tree

Custom implementations of a binary tree.

Category: Custom Implementation

A simple binary tree is often implemented with <set>, <map>, or <priority_queue>. If you still want to implement one by hand, here are two ways:

1. Array-Based (Implicit Tree)

Use a simple <vector>. Move through the tree using index math: i * 2 for left child, and i * 2 + 1 for right child (1-indexed). Inserting via push_back() keeps it balanced.

2. Node-Based (Explicit Pointers)

// Basic Node structure with pointers
class Node {
public:
    int key_value;
    Node* left;
    Node* right;
    
    Node(int val) : key_value(val), left(nullptr), right(nullptr) {}
};
