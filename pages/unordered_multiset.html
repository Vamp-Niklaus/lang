Unordered Multiset

Hash table that allows duplicate keys.

Category: Unordered Container

Overview

#include <unordered_set>
std::unordered_multiset<int> ums;

ums.insert(5);
ums.insert(5);

ums.count(5); // Returns 2

auto range = ums.equal_range(5);
// Elements are not sorted, but identical elements are guaranteed to be adjacent.
