Unordered Multiset

Hash table that allows duplicate keys.

Category: Unordered Container

Overview

#include <unordered_set>
std::unordered_multiset<int> ums;

ums.insert(5);
ums.insert(5);

ums.count(5); // Returns 2

auto range = ums.equal_range(5);
// Elements are not sorted, but identical elements are guaranteed to be adjacent.

Use HashMap<T, Integer> to store frequencies.

collections.Counter

Counter is a dict subclass for counting hashable objects. It is the perfect equivalent to an unordered multiset.

from collections import Counter

c = Counter(['a', 'b', 'b'])
c['a'] += 1
print(c['b']) # 2
