Stack (std::stack)

Adapter template providing a LIFO specialized interface.

Category: Container Adapter

The std::stack template isn't a separate data structure. It is an adapter template, which provides a specialized interface to an existing template (by default, built from deques).

  • Push/Pop/Top: O(1)
#include <stack>

std::stack<int> s;
if (!s.empty()) {
    std::cout << s.size(); 
}

s.push(4);
std::cout << s.top(); // Get the top item

// Note: pop() does NOT return the item that was popped.
// If you want it, you must use top() before pop()
s.pop();

ArrayDeque<T> / Stack<T>

While java.util.Stack exists, it is considered legacy and is synchronized (slower). The officially recommended equivalent for std::stack is using java.util.ArrayDeque.

import java.util.ArrayDeque;
import java.util.Deque;

Deque<Integer> stack = new ArrayDeque<>();

// O(1) Push
stack.push(10);
stack.push(20);

// O(1) Top
int topVal = stack.peek(); // 20

// O(1) Pop
stack.pop(); // Returns and removes 20

list

A standard Python list works perfectly as a stack. Since it's a dynamic array, appending and popping from the end are O(1) operations.

stack = []

# O(1) Push
stack.append(10)
stack.append(20)

# O(1) Top
top_val = stack[-1] # 20

# O(1) Pop
popped = stack.pop() # Returns and removes 20

Array

A standard JavaScript Array provides built-in push and pop methods that operate in O(1) time at the end of the array.

let stack = [];

// O(1) Push
stack.push(10);
stack.push(20);

// O(1) Top
let topVal = stack[stack.length - 1]; // 20

// O(1) Pop
let popped = stack.pop(); // Returns and removes 20
