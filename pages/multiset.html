Multiset (std::multiset)

Set that allows duplicate keys. Balanced binary search tree.

Category: Associative Container

Key Differences from Set

#include <set>
std::multiset<int> ms;

ms.insert(10);
ms.insert(10); // Duplicates allowed!

// count() returns the number of occurrences
int n = ms.count(10); // Returns 2

// erase(val) removes ALL occurrences of val
ms.erase(10); 

// To erase just one occurrence:
ms.erase(ms.find(10));

Range Queries

// Since duplicates exist, equal_range is very useful
auto range = ms.equal_range(10);
// range.first is lower_bound(10)
// range.second is upper_bound(10)
for (auto it = range.first; it != range.second; ++it) {
    // Iterate over all elements equal to 10
}

TreeMap<T, Integer>

Java has no built-in multiset. The standard pattern is to use a TreeMap mapping the element to its frequency count. (Guava provides a TreeMultiset if third-party libraries are allowed).

TreeMap<Integer, Integer> multiset = new TreeMap<>();

// Insert (Increment count)
multiset.put(10, multiset.getOrDefault(10, 0) + 1);

// Erase one instance
int count = multiset.get(10);
if (count == 1) {
    multiset.remove(10);
} else {
    multiset.put(10, count - 1);
}

collections.Counter

To simulate a multiset, you use a Counter which maps elements to their frequency.

from collections import Counter

multiset = Counter()

# Insert (O(1))
multiset[10] += 1
multiset[10] += 1
multiset[20] += 1

# Erase one instance
if multiset[10] > 0:
    multiset[10] -= 1
    if multiset[10] == 0:
        del multiset[10]

# Total elements
total = sum(multiset.values())

Map as Frequency Counter

In JavaScript, you simulate a multiset by mapping elements to their count.

const multiset = new Map();

function insert(val) {
    multiset.set(val, (multiset.get(val) || 0) + 1);
}

function eraseOne(val) {
    if (multiset.has(val)) {
        let count = multiset.get(val);
        if (count === 1) multiset.delete(val);
        else multiset.set(val, count - 1);
    }
}

insert(10);
insert(10);
eraseOne(10);
