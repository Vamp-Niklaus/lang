Queue (std::queue)

Implemented as a deque (double-ended queue) by default. FIFO structure.

  • Enqueue/Dequeue/Front: O(1)
#include <queue>

std::queue<int> q;
q.push(10); // Enqueue an element - O(1)
q.pop();    // Dequeue an element - O(1)
q.front();  // Get the front element - O(1)
q.size();   // Get the number of elements - O(1)
q.empty();  // Check if the queue is empty - O(1)

Queue<T> (ArrayDeque)

In Java, Queue is an interface. You typically instantiate it using an ArrayDeque (faster) or a LinkedList.

import java.util.ArrayDeque;
import java.util.Queue;

Queue<Integer> q = new ArrayDeque<>();

// O(1) Push
q.offer(10); // or q.add(10)

// O(1) Front
int frontVal = q.peek();

// O(1) Pop
q.poll(); // Returns and removes 10

collections.deque

Do NOT use a standard list for a queue, as pop(0) is O(N). Instead, use collections.deque which provides O(1) pop from the left.

from collections import deque

q = deque()

# O(1) Push
q.append(10)

# O(1) Front
front_val = q[0]

# O(1) Pop
popped = q.popleft()

Array (Warning: O(N) Pop)

JavaScript lacks a built-in Queue. Using an Array with push() and shift() works, but shift() is an O(N) operation. For high-performance needs, you must build a custom linked-list queue.

let q = [];

// Push
q.push(10);

// Front
let frontVal = q[0];

// Pop (O(N) operation!)
let popped = q.shift();
