Queue (std::queue)

Implemented as a deque (double-ended queue) by default. FIFO structure.

  • Enqueue/Dequeue/Front: O(1)
#include <queue>

std::queue<int> q;
q.push(10); // Enqueue an element - O(1)
q.pop();    // Dequeue an element - O(1)
q.front();  // Get the front element - O(1)
q.size();   // Get the number of elements - O(1)
q.empty();  // Check if the queue is empty - O(1)

🌎 Polyglot Equivalents

  • ☕
    Java: java.util.Queue<T> (Interface) implemented by LinkedList or ArrayDeque
  • 🐍
    Python: queue.Queue or collections.deque (preferred for performance)
  • 🟨
    JavaScript: Array (using push() and shift()) - note: shift() is O(N).
