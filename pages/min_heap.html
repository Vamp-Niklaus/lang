Min Binary Heap

Implemented using standard vector and algorithm's heap functions.

Category: Custom Implementation

By default, C++ implements max binary heaps. For a Min Binary Heap, there is no direct base container unless you pass a custom comparator to std::priority_queue, or you can implement it by hand using <algorithm> and std::greater<int>().

#include <vector>
#include <algorithm>

std::vector<int> minHeap;
std::make_heap(minHeap.begin(), minHeap.end(), std::greater<int>());

// To add a value
minHeap.push_back(x);
std::push_heap(minHeap.begin(), minHeap.end(), std::greater<int>());

// To remove the min value
std::pop_heap(minHeap.begin(), minHeap.end(), std::greater<int>());
minHeap.pop_back();
