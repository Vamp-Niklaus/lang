Min Binary Heap

Implemented using standard vector and algorithm's heap functions.

Category: Custom Implementation

By default, C++ implements max binary heaps. For a Min Binary Heap, there is no direct base container unless you pass a custom comparator to std::priority_queue, or you can implement it by hand using <algorithm> and std::greater<int>().

#include <vector>
#include <algorithm>

std::vector<int> minHeap;
std::make_heap(minHeap.begin(), minHeap.end(), std::greater<int>());

// To add a value
minHeap.push_back(x);
std::push_heap(minHeap.begin(), minHeap.end(), std::greater<int>());

// To remove the min value
std::pop_heap(minHeap.begin(), minHeap.end(), std::greater<int>());
minHeap.pop_back();

PriorityQueue<T>

Java's PriorityQueue is a Min-Heap by default. No Comparator is needed.

import java.util.PriorityQueue;

PriorityQueue<Integer> minHeap = new PriorityQueue<>();

// O(log N) Push
minHeap.add(50);
minHeap.add(10);
minHeap.add(30);

// O(1) Top (Smallest element)
System.out.println(minHeap.peek()); // 10

// O(log N) Pop
minHeap.poll(); // removes 10

heapq

Python's heapq operates on standard lists and is a Min-Heap by default.

import heapq

min_heap = []

# O(log N) Push
heapq.heappush(min_heap, 50)
heapq.heappush(min_heap, 10)
heapq.heappush(min_heap, 30)

# O(1) Top (Smallest)
print(min_heap[0]) # 10

# O(log N) Pop
smallest = heapq.heappop(min_heap)

Custom Min-Heap Class

Because JavaScript has no built-in heap, you must implement the binary heap logic yourself for O(log N) performance.

class MinHeap {
    constructor() { this.data = []; }
    push(val) {
        this.data.push(val);
        this.bubbleUp(this.data.length - 1);
    }
    pop() {
        if (this.data.length === 1) return this.data.pop();
        const min = this.data[0];
        this.data[0] = this.data.pop();
        this.bubbleDown(0);
        return min;
    }
    bubbleUp(i) {
        while (i > 0) {
            let p = Math.floor((i - 1) / 2);
            if (this.data[i] >= this.data[p]) break;
            [this.data[i], this.data[p]] = [this.data[p], this.data[i]];
            i = p;
        }
    }
    bubbleDown(i) {
        // Implementation omitted for brevity...
    }
}
