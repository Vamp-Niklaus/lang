Deque (std::deque)

Double-ended queue. Similar to vector but efficient at both ends.

Category: Sequence Container

Deques offer a similar interface to vectors, but additionally provide efficient insertion and deletion at the front of the data structure. However, be aware that deques are not guaranteed to be implemented internally as contiguous arrays.

#include <deque>
std::deque<int> d;

d.push_back(10);
d.push_front(5);  // O(1) front insertion

d.pop_back();
d.pop_front();    // O(1) front deletion

ArrayDeque<T>

java.util.ArrayDeque provides dynamic array-backed double-ended queue capabilities. It is generally faster than LinkedList when used as a stack or queue, and provides amortized O(1) operations at both ends.

import java.util.ArrayDeque;
import java.util.Deque;

Deque<Integer> dq = new ArrayDeque<>();

// Insert at both ends O(1)*
dq.addLast(20);  // same as push_back
dq.addFirst(10); // same as push_front

// Access ends O(1)
int first = dq.peekFirst();
int last = dq.peekLast();

// Remove ends O(1)
dq.pollFirst();  // same as pop_front
dq.pollLast();   // same as pop_back

collections.deque

Python's deque is implemented as a doubly-linked list of blocks. It excels at fast O(1) appends and pops from both ends, making it the perfect equivalent for C++ deque/list operations.

from collections import deque

dq = deque()

# Insert O(1)
dq.append(20)      # push_back
dq.appendleft(10)  # push_front

# Access O(1)
first = dq[0]
last = dq[-1]

# Remove O(1)
dq.popleft()       # pop_front
dq.pop()           # pop_back

Array (Using shift/unshift)

JavaScript does not have a dedicated deque. You use the standard Array object. However, note that shift() and unshift() (operating at the front) are O(n) operations because the array memory must be re-indexed.

let dq = [];

// Insert
dq.push(20);      // push_back (O(1))
dq.unshift(10);   // push_front (O(n) - Be careful!)

// Access
let first = dq[0];
let last = dq[dq.length - 1];

// Remove
dq.shift();       // pop_front (O(n) - Be careful!)
dq.pop();         // pop_back (O(1))
