Unordered Set (std::unordered_set)

Implemented as a hash table.

  • Insert/Delete/Search: Average: O(1), Worst: O(n)
#include <unordered_set>

std::unordered_set<int> us;
us.insert(1); // Insert an element - Average: O(1), Worst: O(n)
us.find(1);   // Find an element - Average: O(1), Worst: O(n)
us.erase(1);  // Remove an element - Average: O(1), Worst: O(n)
us.clear();   // Remove all elements - O(n)
us.size();    // Get the number of elements - O(1)

HashSet<T>

java.util.HashSet is the exact equivalent of std::unordered_set. It provides average O(1) time complexity for basic operations by using a HashMap under the hood.

import java.util.HashSet;

HashSet<Integer> set = new HashSet<>();

// O(1) average
set.add(10);

// O(1) average
boolean exists = set.contains(10);

// O(1) average
set.remove(10);

set

Python's built-in set is a highly optimized hash table, offering average O(1) insertion, lookup, and deletion.

s = set()

# Insert
s.add(10)

# Search
if 10 in s:
    print("Found")
    
# Remove
s.remove(10) # Raises KeyError if not found
s.discard(10) # Safe remove

Set

JavaScript's built-in Set is a hash set that maintains insertion order while providing O(1) average time complexities for lookups.

const s = new Set();

// Insert
s.add(10);

// Search
let hasTen = s.has(10);

// Remove
s.delete(10);
