Sets

Unique item collections protecting against duplicate values.

C++
#include <unordered_set>
#include <set>

// Hash set
std::unordered_set<int> s;
s.insert(10);
s.insert(10); // Ignored duplicate

// Sorted Tree balanced set
std::set<int> sortedSet;

// Presence check counter metric
bool found = s.count(10);
