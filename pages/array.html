Array (std::array)

Static contiguous array. Size must be known at compile time.

Category: Sequence Container

Overview

#include <array>

std::array<int, 5> arr = {1, 2, 3, 4, 5};

arr[0] = 10;           // Direct access
arr.at(1) = 20;        // Bounds-checked access
arr.front();           // First element
arr.back();            // Last element
arr.fill(0);           // Fill all with 0

Primitive Arrays

In Java, arrays are fixed-size collections of elements of the same type. They are highly efficient but lack dynamic resizing capabilities. Once created, their length cannot be changed.

Key Characteristics

  • Contiguous memory allocation
  • O(1) access by index
  • Fixed size (.length)
  • Stores primitives or object references

Time Complexities

  • Access (Index)O(1)
  • SearchO(n)
  • Insert / DeleteN/A
// 1. Declaration and Initialization
int[] arr = new int[5];        // Default initialized to 0
int[] nums = {10, 20, 30};     // Initialized with values

// 2. Access and Modify
arr[0] = 5;
int x = nums[1];               // x = 20

// 3. Iteration
for (int i = 0; i < nums.length; i++) {
    System.out.println(nums[i]);
}

// Enhanced for-loop
for (int num : nums) {
    System.out.println(num);
}

// 4. Utility operations
java.util.Arrays.sort(nums);
java.util.Arrays.fill(arr, 9); // Fills arr with 9s

Tuple / Array Module

In Python, a standard list is actually a dynamic array (like std::vector). For a strictly fixed-size, immutable array, use a tuple. For typed, contiguous memory C-style arrays, use the array module.

# 1. Tuples (Immutable, fixed-size)
tup = (10, 20, 30)
# tup[0] = 15  # TypeError: tuple object does not support item assignment

# 2. The 'array' module (Typed contiguous array)
import array
arr = array.array('i', [1, 2, 3, 4, 5]) # 'i' specifies signed integer

arr[0] = 10
print(arr[0]) # 10

# Note: array.array allows append(), making it dynamic, 
# but restricts types strictly unlike standard Python lists.

Typed Arrays

In JavaScript, standard Arrays are dynamic and heterogenous. To get true C++ std::array behavior (fixed-size, typed, contiguous memory), you use Typed Arrays like Int32Array, Float64Array, etc.

// 1. Declaration
const arr = new Int32Array(5); // Fixed size of 5 integers, initialized to 0
const nums = new Int32Array([10, 20, 30]);

// 2. Access and Modify
arr[0] = 5;
console.log(nums[1]); // 20

// 3. Size is fixed
// arr.push(10); // TypeError: arr.push is not a function

// 4. Iteration
for (let num of nums) {
    console.log(num);
}

// 5. Using ArrayBuffer directly
const buffer = new ArrayBuffer(16); // 16 bytes
const view = new Int32Array(buffer); // Maps to 4 integers
