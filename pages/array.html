Array (std::array)

Static contiguous array. Size must be known at compile time.

Category: Sequence Container

Overview

#include <array>

std::array<int, 5> arr = {1, 2, 3, 4, 5};

arr[0] = 10;           // Direct access
arr.at(1) = 20;        // Bounds-checked access
arr.front();           // First element
arr.back();            // Last element
arr.fill(0);           // Fill all with 0
