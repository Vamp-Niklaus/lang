Graph / N-Aries Tree

Implementations for graph traversal and representation.

Category: Custom Implementation

1. Implicit Graph (Indexed 0 to N)

When nodes are identified using indices, the vertex matches the vector index (Adjacency List).

// Unweighted Graph
std::vector<std::vector<int>> graph;

// Weighted Graph (Edge weights)
std::vector<std::vector<std::pair<int, int>>> graph;

2. Object-Oriented Graph

When nodes require custom identifiers or properties, you must create dedicated classes.

class Edge;

class Vertex {
public:
    int id;
    std::vector<Edge*> ptr_edges;
};

class Edge {
public:
    Vertex* from;
    Vertex* to;
    int cost;
};

class Graph {
public:
    std::vector<Vertex> vertices;
};

Adjacency List

Represented using List<List<Integer>> or Map<Integer, List<Integer>>.

Map<Integer, List<Integer>> adj = new HashMap<>();
adj.putIfAbsent(1, new ArrayList<>());
adj.get(1).add(2);

Adjacency List

Represented using a dict of lists or collections.defaultdict(list).

from collections import defaultdict
adj = defaultdict(list)
adj[1].append(2)

Adjacency List

Represented using a Map or an Object.

const adj = {};
if (!adj[1]) adj[1] = [];
adj[1].push(2);
