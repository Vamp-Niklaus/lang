Unordered Map (std::unordered_map)

Implemented as a hash table.

  • Insert/Delete/Search by key: Average: O(1), Worst: O(n)
#include <unordered_map>

std::unordered_map<int, int> um;
um[1] = 2;          // Insert or update a key-value pair - Average: O(1), Worst: O(n)
um.emplace(1, 2);   // Insert a key-value pair - Average: O(1), Worst: O(n)
um.find(1);         // Find an element by key - Average: O(1), Worst: O(n)
um.erase(1);        // Remove an element by key - Average: O(1), Worst: O(n)
um.clear();         // Remove all elements - O(n)
um.size();          // Get the number of elements - O(1)

HashMap<K, V>

java.util.HashMap is the standard hash table based implementation of the Map interface, identical in purpose to std::unordered_map.

import java.util.HashMap;

HashMap<String, Integer> map = new HashMap<>();

// O(1) average
map.put("A", 1);

// O(1) average
int val = map.get("A");

// O(1) average
map.remove("A");

dict

Python's built-in dictionary dict is an immensely optimized hash map that also happens to preserve insertion order (since Python 3.7).

d = {}

# O(1) Insert
d["A"] = 1

# O(1) Search
if "A" in d:
    val = d["A"]
    
# O(1) Delete
del d["A"]

Map / Object

In JavaScript, you can use a plain Object for string keys, or a Map for keys of any type (including objects).

const map = new Map();

// O(1) Insert
map.set("A", 1);

// O(1) Search
let val = map.get("A");

// O(1) Delete
map.delete("A");
