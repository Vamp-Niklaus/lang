Vector (std::vector)

Implemented as a dynamic array. Dynamically-sized, heap-based array without explicit memory management.

Category: Sequence Container

Performance

  • Access: O(1)
  • Insert/Delete at end: O(1) amortized
  • Insert/Delete elsewhere: O(n)

Constructors

  • vector<int> v; (Default: empty)
  • vector<int> v2(v); (Copy constructor)
  • vector<int> v(4, -25); (4 elements, value -25)
  • vector<int> v(4); (4 elements, default 0)

Reserving Space

Optimize performance by pre-allocating space to reduce reallocations.

std::vector<int> v;
v.reserve(50); // Pre-allocate space for 50 items

Array Conversion

Vectors are contiguous, so you can pass them to C-style array functions.

void f(int *a);
std::vector<int> v;
f(&v[0]);

Iteration

// Standard Iterator
for (std::vector<int>::iterator it = v.begin(); it != v.end(); ++it) {
    *it = 5;
}

// Reverse Iterator
for (std::vector<int>::reverse_iterator it = v.rbegin(); it != v.rend(); ++it) { ... }

// Const Iterator (Required if vector is passed as const reference)
for (std::vector<int>::const_iterator it = v.begin(); it != v.end(); ++it) { ... }

Core Operations

v.push_back(10);    // Insert at the end
v.pop_back();       // Remove the last element

v.insert(v.begin() + 3, 22); // Insert 22 at position 3
v.insert(v.end() - 2, 55);   // Insert 55 at third-to-last position

v.erase(v.begin() + 3);      // Remove item at position 3
v.erase(v.begin() + 10, v.begin() + 15); // Remove items 10 .. 15 (exclusive)

ArrayList<T>

java.util.ArrayList is the exact equivalent of std::vector. It is a resizable array implementation of the List interface. It doubles in size when capacity is reached (amortized O(1) insertion).

Method Mapping

  • push_back(val) → add(val)
  • pop_back() → remove(size - 1)
  • operator[i] → get(i) / set(i, val)
  • size() → size()

Time Complexities

  • Access get(i)O(1)
  • Add at endO(1)*
  • Insert at indexO(n)
import java.util.ArrayList;
import java.util.Collections;

ArrayList<Integer> vec = new ArrayList<>();

// Add elements (Amortized O(1))
vec.add(10);
vec.add(20);

// Insert at index (O(n))
vec.add(0, 5);

// Access and Modify (O(1))
int val = vec.get(1);
vec.set(1, 15);

// Remove last element (O(1))
vec.remove(vec.size() - 1);

// Iteration
for (int num : vec) {
    System.out.println(num);
}

// Sorting
Collections.sort(vec);

List

The standard Python list is implemented under the hood as a dynamic array of pointers. It grows automatically when full and provides amortized O(1) appends, exactly like a C++ vector.

vec = []

# Append elements (Amortized O(1))
vec.append(10)
vec.append(20)

# Insert at index (O(n))
vec.insert(0, 5)

# Access and Modify (O(1))
val = vec[1]
vec[1] = 15

# Remove last element (O(1))
last_val = vec.pop()

# Iteration
for num in vec:
    print(num)
    
# Sorting (In-place)
vec.sort()

Array

The standard JavaScript Array functions as a dynamic array. Modern V8 engines optimize continuous arrays of the same type to be contiguous memory blocks, behaving identically to std::vector.

let vec = [];

// Append elements (O(1))
vec.push(10);
vec.push(20);

// Insert at index (O(n))
vec.splice(0, 0, 5);

// Access and Modify (O(1))
let val = vec[1];
vec[1] = 15;

// Remove last element (O(1))
let lastVal = vec.pop();

// Iteration
for (let num of vec) {
    console.log(num);
}

// Map / Filter
let doubled = vec.map(x => x * 2);
